## VSCode Devcontainer Config Files

This folder contains the necessary configs for running this project in a devcontainer. Devcontainers offer an isolated environment from your host computer, analogous to a virtual machine. This lets you use fresh installs of any operating system you choose, and keeps whatever you do/run/install separate from your host environment.
